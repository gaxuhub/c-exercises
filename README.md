# c-exercises
This is a sequence of C programs that tries to use all functions that I've learned.
1. Create the type AlunoVC, with RA (int), nome (char[12]) and media (double).
Create the type AlunoPC, with RA (int), nome (char * ) and media (double).
Write a program that printsout the size in bytes for each of the following types:
char, int, float, double, AlunoVC and AlunoPC.
3. Faça um programa que solicite do usuário o tamanho do vetor a ser lido, 
faça a alocação dinâmica desse vetor com o tipo char e solicite que o usuário
digite uma string. Por fim, imprima a string digitada.
4. Faça um programa que solicite do usuário um inteiro positivo N
e aloque um vetor de inteiros com tamanho N. Preencha o vetor com valores que
correspondem a 2 elevado ao índice de cada posição
no vetor. Por fim, imprima o vetor.
5. Crie um registro chamado Nodo, que contém um inteiro
e um ponteiro para outro Nodo. Faça um
programa em que o usuário é capaz de escolher entre: 1) criar um nodo; 2) imprimir os valores de
todos os nodos; 3) sair do programa. O programa deve iniciar com uma variável cujo tipo é um
ponteiro para um Nodo. Essa variável inicia apontando para NULL. Ao criar o primeiro Nodo, essa
variável deve apontar para ele. Cada Nodo, a partir do primeiro, aponta para o Nodo seguinte.
6. Acrescente no programa anterior a possibilidade de
apagar um Nodo (questão anterior). Lembre-se
que ao apagar um Nodo, o Nodo anterior deverá apontar para o Nodo seguinte ao que está sendo
apagado.
7. Crie um programa que solicita do usuário 3 números 
float e grava esses números em um arquivo binário.
8. Crie um programa que lê os números do arquivo binário gravado na questão anterior
e os imprime na tela.
9. Crie o tipo Aluno com nome (char*) e RA (int). Crie um programa que utiliza o tipo 
Aluno e permite, por meio de um menu de opções, incluir, excluir e visualizar alunos em um 
vetor. Para cada ação (incluir, excluir e visualizar) realizada, o programa deve gravar
em um arquivo de texto uma mensagem de log. Por exemplo, se um aluno com nome JOAO e RA 6532
foi incluído no vetor, o programa deve gravar no arquivo uma mensagem 
( ALUNO INCLUIDO: JOAO - 6532 ). Ao
escolher sair do programa, todo o arquivo de log deve ser lido e impresso na tela.
10. Crie um registro Ponto com as coordenadas x e y (float).
Crie um programa que possui um vetor do tipo Ponto. O programa deve apresentar para o usuário
um menu em que é possível escolher entre fechar o programa, adicionar um Ponto no vetor ou
visualizar todos os Pontos do arquivo. Quando ele escolher sair, o programa deve gravar todos
os Pontos contidos no vetor em um arquivo.
11. Complemente o programa da questão anterior incluindo a capacidade de ler todos os pontos
armazenados no arquivo e adicioná-los no vetor de Pontos assim que o programa é ligado. Adicione
também no menu a opção de apagar um ponto do vetor.
